Restler Boot

A Micro-framework for building REST+JSON API with emphasis on Domain Driven Design methodology and NoSQL persistence.

Overview

The easiest way to build a RESTful JSON API using PHP.

Features

  • Develop reusable Microservices.
  • Built with Domain Driven Design style
  • HATEOAS compliant
  • NoSQL enabled for BigData capability

Installation

The framework can be installed via Composer. Simply execute the following command: {% highlight bash %} composer create-project buonzz/restler-boot myapi {% endhighlight %}
